## 핵심 키워드 정리
- **while문**: 조건식이 참(`true`)인 동안 계속해서 코드를 실행
- **do-while문**: 조건에 상관없이 무조건 **최소 한 번**은 실행한 뒤 조건을 검사
- **for문**: 반복 횟수가 정해져 있을 때 사용하는 가장 정돈된 형태의 반복문
- **break & continue**: 반복의 흐름을 강제로 종료하거나 다음 순서로 건너뜁니다.
- **중첩 반복문**: 반복문 안에 반복문을 넣어 다차원적인 작업을 수행합니다.

## Java
### 1. while문 vs for문: 언제 무엇을 쓸까?
**while**: "운동장 3바퀴 돌아!" 보다는 "지칠 때까지 돌아"에 가깝습니다. (조건 중심)
**for**: "운동장 딱 3바퀴만 돌아!" 처럼 횟수가 명확할 때 유리합니다. (횟수 중심)

**Java 코드 비교**
```java
// while문: 변수 선언, 증감식이 분산되어 있음
int i = 1;
while (i <= 3) {
	sum += i;
	i++;
}

// for문: 초기식, 조건식, 증감식이 한 줄에 있어 가독성이 좋음
for (int j = 1; j <= 3; j++) {
	sum += j;
}
```
**요약**: 반복 횟수가 정해져 있고 카운터 변수가 중요하다면 **for문**이 구조적으로 훨씬 깔끔하고 유지보수에 좋습니다.

### 2. 반복의 흐름 제어: break와 continue
**break**: "비상구"와 같습니다. 특정 조건을 만나는 즉시 반복문을 탈출합니다.
**continue**: "이번 판만 패스!"와 같습니다. 현재 루프의 나머지 코드를 건너뛰고 다음 반복(조건 검사 또는 증감식)으로 이동합니다.

```java
while (i <= 5) {
	if (i == 3) {
		i++;
		continue;  // 3일 때는 출력하지 않고 다음 루프로 점프! 
	}
	System.out.println(i);
	i++;
}
```
### 3. 중첩 반복문 (Nested Loop)
반복문 내부에 또 다른 반복문을 사용하는 구조입니다. 외부 루프가 1번 실행될 때마다 내부 루프는 설정된 횟수만큼 전체 실행됩니다.
